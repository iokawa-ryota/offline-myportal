# リンク集 & 便利ツール

シンプルなリンク集と便利ツールが 1 つの HTML ファイルで使えるページです。

## 特徴

- **1 ファイル完結**: `index.html` 1 つだけ
- **外部依存なし**: インターネット接続不要で動作（リンク先除く）
- **軽量**: CSS/JavaScript をインライン化
- **ダブルクリックで起動**: ブラウザで直接開けます

## 使い方

1. `index.html` をダブルクリック
2. ブラウザで開きます

## 機能

### ✅ TODOリスト

- やることの追加・修正・完了チェック・削除
- 修正時は該当行に入力ボックスを表示してその場で編集
- Enter キーで素早く追加
- 完了タスクは通常リストから非表示（ボタンで表示切替）
- ブラウザの LocalStorage に自動保存

### 📚 リンク集

- よく使う Web サイトへのクイックアクセス
- カスタマイズ可能

### 🛠️ 便利ツール

- **文字数カウンター**: リアルタイムで文字数・行数をカウント
- **Base64 エンコード/デコード**: テキストと Base64 の相互変換
- **タイムスタンプ変換**: Unix タイムスタンプと日時の変換
- **URL エンコード/デコード**: URL 文字列のエンコード/デコード
- **カラーコード変換**: HEX と RGB の表示

## カスタマイズ

`index.html` を直接編集してリンクやツールを追加できます。

### リンクの追加例

```html
<a href="https://example.com" class="link-card" target="_blank">
  <h3>🌐 サイト名</h3>
  <p>説明</p>
</a>
```

## 動作環境

- モダンブラウザ（Chrome、Firefox、Edge、Safari 等）
- サーバー不要
- インターネット接続不要（外部リンク先へのアクセスを除く）
